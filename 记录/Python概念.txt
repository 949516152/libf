1. Python 概念
  1.1Python的关键要素
	1）要素1
		数据类型
		String、Number
		
		Python所能表示的整数大小只受限于机器内存而非固定数量的字节数。
		Python使用方括号（ [] ）来存取字符串等序列的某一项;
		Python语法中索引是从 0 开始的;
		在Python中，字符就是长度为1的字符串
		如果将一种数据项从一种数据类型转换为另一种数据类型，可以使用语法：datatype(item);
		其中，int()方法允许转换的字符串头尾有空格。
		
	2）要素2
		对象引用    在Python中没有存储某种数据类型的变量，而是使用“对象引用”
			赋值操作符 “=”
			注释符 “#”
			Python标识符，是由数字、字母、下划线组成，不能以数字开头，不能与关键字相同；
				标识符没有长度限制，对大小写敏感
			
			Python的“动态类型”机制，
				即在任何时候，只要需要，某个对象引用都可以重新引用一个不同的对象引用，亦可以是不同的数据类型
			
			type()函数会返回给定数据项的数据类型
	
	3) 要素3 
		组合数据类型  tuple、List、Set、Dictionary
		元组和列表可以存放任意数量、任意类型的数据项;
			其中，元组是固定的，不可修改的，由小括号 () 构成，只有一个数据项的元组，在数据项后必须加 ，逗号，
				使用  ()  表示空元组;
				  列表是可修改的,由 方括号[]构成;
		输出时，使用逗号分隔参数或默认形成 元组，
			如果将元组常值作为参数传递时，必须使用小括号 () 进行封装，避免混淆。
			
			实质上，元组和列表并不真正存放储存数据，而是对象引用。
			
			元组、列表和字符串等数据类型是有大小的。
			
			点（“存取属性”）操作符用于存取对象的某个属性。
	4)元素4
		逻辑操作符
			身份操作符  is   not is
				is 操作符只需要对对象所在的内存地址进行比较;
				最常见使用is的情况是将数据项与内置的空对象None进行比较，None通常作为位置标记值，指示“未知”或不存在
			身份操作符的作用是，查看两个对象引用是否指向相同的对象，或者查看某个对象是否为None；
				比较对象值应使用比较运算符
				
				
			比较操作符 == <  >   <=   >=    !=
				比较操作符对对象值进行比较，即对象引用所指向的具体对象。
			Python的比较操作符可以进行结链比较
			Python对无意义的比较操作会抛异常，例如类型不一致，可以先转换在比较
