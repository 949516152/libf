本包学习工厂方法模式：
    工厂方法模式：
        提供一个用于创建对象的接口（工厂接口），让其实现类（工厂实现类）决定实例化哪一个类，并由该实现类创建对应类的实例。

    模式作用：
        可以在一定程度上解耦；
        可以增加一定的拓展性；
        可以一定程度增加代码的封装性、可读性；

        抽象工厂才是实际意义的工厂模式，工厂方法只是抽象工厂的一个比较常见的情况。

    模式要素：
        提供一个产品类接口，产品类都实现这个接口，也可以是抽象类
        提供一个工厂类接口，工厂类都是实现这个工厂，
        由工厂实现类创建产品的实例。工厂实现类应有一个方法，用来实例化产品。


    工厂方法模式：
        该模式提供工厂接口工厂类都要实现这个工厂接口
